<p>Crawlee for Python v0.5 is now available! This is our biggest release to date, bringing new ported functionality from the <a href="https://github.com/apify/crawlee" rel="noopener noreferrer" target="_blank">Crawlee for JavaScript</a>, brand-new features that are exclusive to the Python library (for now), a new consolidated package structure, and a bunch of bug fixes and further improvements.</p>

<h2 class="anchor anchorTargetHideOnScrollNavbar_vjPI" id="new-package-structure">New package structure<a class="hash-link" href="https://crawlee.dev/blog/crawlee-for-python-v05#new-package-structure" title="Direct link to New package structure">​</a></h2>
<p>We have introduced a new consolidated package structure. The goal is to streamline the development experience, help you find the crawlers you are looking for faster, and improve the IDE's code suggestions while importing.</p>

<h2 class="anchor anchorTargetHideOnScrollNavbar_vjPI" id="continued-parity-with-crawlee-js">Continued parity with Crawlee JS<a class="hash-link" href="https://crawlee.dev/blog/crawlee-for-python-v05#continued-parity-with-crawlee-js" title="Direct link to Continued parity with Crawlee JS">​</a></h2>
<p>We are constantly working toward feature parity with our JavaScript library, <a href="https://github.com/apify/crawlee" rel="noopener noreferrer" target="_blank">Crawlee JS</a>. With v0.5, we have brought over more functionality:</p>
<h3 class="anchor anchorTargetHideOnScrollNavbar_vjPI" id="html-to-text-context-helper">HTML to text context helper<a class="hash-link" href="https://crawlee.dev/blog/crawlee-for-python-v05#html-to-text-context-helper" title="Direct link to HTML to text context helper">​</a></h3>

<h3 class="anchor anchorTargetHideOnScrollNavbar_vjPI" id="use-state">Use state<a class="hash-link" href="https://crawlee.dev/blog/crawlee-for-python-v05#use-state" title="Direct link to Use state">​</a></h3>
<p>The <a href="https://www.crawlee.dev/python/api/class/UseStateFunction" rel="noopener noreferrer" target="_blank"><code>use_state</code></a> crawling context helper makes it simple to create and manage persistent state values within your crawler. It ensures that all state values are automatically persisted. It enables you to maintain data across different crawler runs, restarts, and failures. It acts as a convenient abstraction for interaction with <a href="https://www.crawlee.dev/python/api/class/KeyValueStore" rel="noopener noreferrer" target="_blank"><code>KeyValueStore</code></a>.</p>
